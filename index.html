What is Modflux?

Modflux is not an NFT collection.

It is a consumable execution layer for AI agents.

Each Modflux NFT represents prepaid execution capacity. When Sentinel performs real work, one execution charge is consumed on-chain.

When charges reach zero, the NFT is automatically burned.

No burn means no execution.

How it works

  1. Mint a Modflux NFT
  2. Trigger Sentinel to perform a task
  3. Sentinel executes real work
  4. One charge is burned on-chain
  5. When charges reach zero, the NFT is destroyed

Execution is only allowed when a charge is consumed.

Why Modflux exists

AI agents fail when execution is free.

Free compute leads to abuse, spam, and unreliable systems.

  • No free execution
  • No subscriptions
  • No hidden costs
  • Verifiable, on-chain proof of work

If Sentinel runs, someone paid for it.
